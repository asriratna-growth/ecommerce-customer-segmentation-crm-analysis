# Ecommerce Customer Segmentation, Retention & CRM Analysis

## Project Overview

This project analyzes customer purchasing behavior from a UK-based online retail dataset using SQL, BigQuery, and Looker Studio.

The objective is to identify high-value customers, understand retention risks, estimate customer lifetime value (CLV), and generate actionable CRM recommendations through RFM segmentation.

---

## Business Questions

This project answers:

- Which customer segments generate the most revenue?
- Who are the most valuable customers?
- Which customers are at risk of churn?
- What is the estimated Customer Lifetime Value (CLV) by segment?
- What CRM strategies should be applied to each segment?

## Dataset

### Source

UCI Online Retail Dataset: https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset

Note:
This project was created for educational and portfolio purposes only. All rights remain with the original dataset author.

### Period

December 2010 – December 2011

### Description

The dataset contains transactional records from a UK-based online retailer.

Key fields include:

- Invoice Number
- Customer ID
- Invoice Date
- Product Description
- Quantity
- Unit Price
- Revenue

## Key Findings

- Champions generate approximately 65% of total portfolio CLV.
- Loyal Customers represent the second most valuable customer segment.
- 40% of customers are currently categorized as retention risk.
- At Risk customers contribute significant revenue and should be prioritized for win-back campaigns.
- Potential Loyalists represent the strongest growth opportunity for future CLV expansion.
